Payroll impact: Paid days, LOP, overtime, bonus and manual deductions directly affect payroll calculation.

Required File Format

Your Excel/CSV file must contain the minimum required columns. Column names should be exactly the same.

Required Columns

emp_code month working_days present_days weekly_off paid_leave holiday lop_days overtime_hours

Optional Columns

paid_days bonus manual_deduction
Month format must be YYYY-MM, for example 2026-12. If paid_days is blank or 0, system calculates it automatically: present_days + weekly_off + paid_leave + holiday - lop_days.
UPLOAD RULES

Attendance Upload Checks

The system checks your file before saving attendance data.

Employee Code Check Every emp_code must exist in Employee Master.
Month Format Month must be in YYYY-MM format, for example 2026-12.
Paid Days Formula Paid days should match present + weekly off + paid leave + holiday - LOP.
Duplicate Check Same employee and same month cannot appear twice in one file.
Numeric Validation Working days, leave, overtime, bonus and deductions must be valid positive numbers.
Replace Existing Month Uploading the same month again replaces old attendance for that month.
Maximum Days Check Working days, present days, leave, holiday and LOP cannot be greater than 31.
Payroll Ready After upload, run payroll for the selected month from Run Payroll page.
