Required File Format

Your Excel/CSV file must contain the following columns. Column names should be exactly the same.

emp_code month working_days present_days overtime_hours bonus manual_deduction
Month format must be YYYY-MM, for example 2026-12.
UPLOAD RULES

Attendance Upload Checks

The system checks your file before saving attendance data.

Employee Code Check Every emp_code must exist in Employee Master.
Month Format Month must be in YYYY-MM format.
Present Days Check Present days cannot be greater than working days.
Duplicate Check Same employee and same month cannot appear twice.
Numeric Validation Working days, overtime, bonus and deductions must be valid numbers.
Replace Existing Month Uploading the same month again will replace old attendance for that month.
