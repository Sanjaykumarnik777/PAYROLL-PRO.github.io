Required File Format

Your Excel/CSV file must contain the minimum required columns. Keep column names exactly the same.

Minimum Required Columns

Recommended Optional Columns

department gender tax_regime other_annual_deductions special_allowance uan_no esic_no bank_name account_no ifsc_code

UPLOAD RULES

Employee Upload Checks

The system checks your employee file before saving master data.

Employee Code Required Every employee must have a unique emp_code.
Duplicate Check Duplicate emp_code in the same file will be blocked.
Salary Validation monthly_salary must be greater than 0.
Gender Format Use male, female or other. Blank value defaults to male.
Tax Regime Use old or new. Blank values will default to new.
Leave Balance Setup Leave balances are automatically created for new employees.
Update Existing Employees If emp_code already exists, employee details are updated instead of duplicated.
Free Plan Limit Only newly added employees count toward the free plan 10 employee limit.
