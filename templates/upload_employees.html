Required File Format

Your Excel/CSV file must contain these columns. Keep column names exactly the same.

emp_code employee_name role department gender monthly_salary tax_regime other_annual_deductions special_allowance uan_no esic_no bank_name account_no ifsc_code
Minimum required columns are emp_code, employee_name, role and monthly_salary.
UPLOAD RULES

Employee Upload Checks

The system checks your employee file before saving master data.

Employee Code Required Every employee must have a unique emp_code.
Duplicate Check Duplicate emp_code in the same file will be blocked.
Salary Validation monthly_salary must be greater than 0.
Gender Format Use male, female or other.
Tax Regime Use old or new. Blank values will default to new.
Leave Balance Setup Leave balances are automatically created for new employees.
